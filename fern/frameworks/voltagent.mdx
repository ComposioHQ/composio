**Composio enables** your **VoltAgent agents** to **connect** with many **tools**!

<Tip>Goal: Star a repository on GitHub with a VoltAgent agent</Tip>

## Star A Repository on Github
In this example, we will use VoltAgent to star a repository on Github using Composio Tools

<Steps>
  <Step title="Create VoltAgent project">
    ```bash title="JavaScript"
    npm create voltagent-app@latest
    ```

    You can get the full VoltAgent installation instructions [here](https://voltagent.dev/docs/quick-start/)
  </Step>


  <Step title="Import Libraries & Setup MCPConfiguration">
    ```javascript title="JavaScript"
    import { Agent, MCPConfiguration } from "@voltagent/core";
    import { VercelAIProvider } from "@voltagent/vercel-ai";
    import { openai } from "@ai-sdk/openai";

    const mcp = new MCPConfiguration({
      servers: {
        github: {
          type: "http",
          url: "https://mcp.composio.dev/github/[private-url-path]",
        },
      },
    });

    ```
  </Step>

  <Step title="Execute the Agent">
    ```javascript title="JavaScript" {8-20}
    const mcp = new MCPConfiguration({
        servers: {
            github: {
            url: new URL("https://mcp.composio.dev/github/[private-url-path]"),
            },
        },
    });

    const agent = new Agent({
      name: "GitHub Starrer Agent",
      description: "You help users star GitHub repositories",
      llm: new VercelAIProvider(),
      model: openai("gpt-4o-mini"),
      tools: await mcpConfig.getTools()
    });

    const stream = await agent.streamText("Please star the repository 'composiohq/composio'");

    for await (const chunk of stream.textStream) {
        console.log(chunk);
    }
    ```
  </Step>
</Steps>
