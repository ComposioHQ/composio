Auth Configs define how authentication works for each toolkit. They specify the authentication method, required credentials, and configuration needed for users to connect their accounts.

## What Are Auth Configs?

An Auth Config is a template that defines:
- **Authentication Method**: OAuth2, API Key, Basic Auth, etc.
- **Required Fields**: What credentials users need to provide
- **Service Configuration**: OAuth app settings, API endpoints, scopes

<CodeGroup>

```python
# Create an auth config for GitHub
auth_config = await composio.auth_configs.create('github', {
    'type': 'use_composio_managed_auth',
    'name': 'GitHub Integration'
})
```
```typescript
// Create an auth config for GitHub
const authConfig = await composio.authConfigs.create('github', {
  type: 'use_composio_managed_auth',
  name: 'GitHub Integration'
});
```

</CodeGroup>

## Auth Config Types

### Composio Managed Auth
Composio handles OAuth applications and credentials for you:

<CodeGroup>


```python
auth_config = await composio.auth_configs.create('github', {
    'type': 'use_composio_managed_auth',
    'name': 'GitHub - Composio Managed'
})
```
```typescript
const authConfig = await composio.authConfigs.create('github', {
  type: 'use_composio_managed_auth',
  name: 'GitHub - Composio Managed'
});
```
</CodeGroup>

**Benefits:**
- No OAuth app setup required
- Composio handles token management
- Quick setup for development and testing
- Built-in security best practices

### User Credentials
Users provide their own API keys or credentials:

<CodeGroup>


```python
auth_config = await composio.auth_configs.create('openai', {
    'type': 'user_credentials',
    'name': 'OpenAI API Key',
    'fields': {
        'api_key': {
            'type': 'string',
            'description': 'Your OpenAI API key',
            'required': True
        }
    }
})
```
```typescript
const authConfig = await composio.authConfigs.create('openai', {
  type: 'user_credentials',
  name: 'OpenAI API Key',
  fields: {
    api_key: {
      type: 'string',
      description: 'Your OpenAI API key',
      required: true
    }
  }
});
```
</CodeGroup>

### Custom Auth
Define custom authentication flows:

<CodeGroup>
```python Python
auth_config = await composio.auth_configs.create('custom_service', {

            'description': 'OAuth redirect URI',
```

## Authentication Schemes

Auth configs support different authentication schemes:

### OAuth2
Most modern services use OAuth2:

<CodeGroup>

```python
# Get required fields for OAuth2 setup
fields = await composio.toolkits.get_auth_config_creation_fields('github', 'OAUTH2')

# Create OAuth2 auth config
auth_config = await composio.auth_configs.create('github', {
    'type': 'custom_auth',
    'name': 'My GitHub App',
    'fields': {
        'client_id': 'your_github_client_id',
        'client_secret': 'your_github_client_secret',
        'scopes': 'repo,user:email'
    }
})
```
```typescript
// Get required fields for OAuth2 setup
const fields = await composio.toolkits.getAuthConfigCreationFields('github', 'OAUTH2');

// Create OAuth2 auth config
const authConfig = await composio.authConfigs.create('github', {
  type: 'custom_auth',
  name: 'My GitHub App',
  fields: {
    client_id: 'your_github_client_id',
    client_secret: 'your_github_client_secret',
    scopes: 'repo,user:email'
  }
});
```

</CodeGroup>

<CodeGroup>

```python
connection = await composio.connected_accounts.initiate(
    'user_123',
    'auth_config_456',  # Specific auth config ID

        api_key='user_provided_key'
    )
)
```

```typescript
// Create connected account using specific auth config
const connection = await composio.connectedAccounts.initiate(
  'user_123',
  'auth_config_456', // Specific auth config ID
  {
    config: AuthScheme.ApiKey({
      api_key: 'user_provided_key'

);
```

## Troubleshooting

### OAuth Redirect Issues
- Ensure redirect URI matches exactly
- Check if the OAuth app is configured correctly
- Verify the redirect URI is accessible

### Missing Fields
```javascript
// Check what fields are required
const fields = await composio.toolkits.getAuthConfigCreationFields('toolkit', 'OAUTH2');
console.log('Required fields:', fields.filter(f => f.required));
```

### Connection Failures
- Verify client ID and secret are correct
- Check OAuth app permissions and scopes
- Ensure the auth config is not disabled

## Security Considerations

### Credential Protection
- Never commit secrets to version control
- Use environment variables or secure secret management
- Rotate credentials regularly

### Scope Management
- Request only necessary permissions
- Use least privilege principle
- Document required scopes clearly

### Access Control
- Limit who can create/modify auth configs
- Use different configs for different environments
- Monitor auth config usage


Auth configs are the foundation of secure authentication in Composio. Understanding how to create and manage them effectively ensures your users can safely connect their accounts to your application.
