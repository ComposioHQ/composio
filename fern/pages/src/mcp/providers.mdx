## Anthropic (Claude)

Use MCP servers with Claude API and Claude Desktop.

<CodeGroup>
<CodeGroupItem title="Python">
```python
# Python SDK - Anthropic integration coming soon
# Use the standard MCP server URL generation for now
instance = server.generate("user@example.com")
print(f"MCP URL for Claude: {instance['url']}")
```
</CodeGroupItem>
<CodeGroupItem title="TypeScript">
```typescript
import Anthropic from '@anthropic-ai/sdk';

const anthropic = new Anthropic({
  apiKey: process.env.ANTHROPIC_API_KEY,
});

// Get MCP server URL
const serverUrls = await composio.mcp.getServer(
  server.id,
  "user@example.com"
);

// Use with Claude (when MCP support is available)
const stream = await anthropic.beta.messages.stream({
  model: "claude-3-5-sonnet-20241022",
  mcp_servers: serverUrls,
  messages: [{
    role: "user",
    content: "Fetch my latest emails"
  }]
});
```
</CodeGroupItem>
</CodeGroup>

## Vercel AI SDK

Use MCP servers with Vercel's AI SDK.

```typescript
import { experimental_createMCPClient as createMCPClient } from 'ai';
import { SSEClientTransport } from '@modelcontextprotocol/sdk/client/sse.js';

// Get server URL
const serverUrls = await composio.mcp.getServer(
  server.id,
  "user@example.com"
);

// Create MCP client
const mcpClient = await createMCPClient({
  name: 'composio-mcp-client',
  transport: new SSEClientTransport(new URL(serverUrls.url)),
});

// Use with Vercel AI
const tools = await mcpClient.tools();
```

## Cursor IDE

Configure MCP servers in Cursor settings.

```json
{
  "mcpServers": {
    "composio": {
      "command": "npx",
      "args": [
        "-y",
        "@modelcontextprotocol/server-compose",
        "YOUR_MCP_SERVER_URL"
      ]
    }
  }
}
```

## Claude Desktop

Add to Claude Desktop configuration:

**macOS**: `~/Library/Application Support/Claude/claude_desktop_config.json`
**Windows**: `%APPDATA%\Claude\claude_desktop_config.json`

```json
{
  "mcpServers": {
    "composio-gmail": {
      "command": "node",
      "args": [
        "/path/to/mcp-server.js",
        "YOUR_MCP_SERVER_URL"
      ]
    }
  }
}
```
