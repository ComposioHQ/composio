Build an agent that summarizes your Gmail inbox.

<Tabs>
<Tab title="OpenAI (Python)">

<Steps>
<Step title="Install Composio SDK">
```bash
pip install composio openai-agents composio-openai-agents
```
</Step>

<Step title="Create session for user">
Sessions are ephemeral, user-scoped and manage connections and tools for that user. See [Session Management](/docs/user-management) and [Configuring Toolkits](/tool-router/configuring-toolkits).

<Info>
`your-api-key`: Your Composio API key from the [dashboard](https://platform.composio.dev/?next_page=/settings).
`pg-user-550e8400-e29b-41d4`: The [ID of the user](/docs/user-management) in your system.
</Info>
```python
from composio import Composio
from composio_openai_agents import OpenAIAgentsProvider

composio = Composio(
    api_key="your-api-key",
    provider=OpenAIAgentsProvider()
)

# Create a session for a user
session = composio.create(user_id="pg-user-550e8400-e29b-41d4")
```
</Step>

<Step title="Run agent">
When the agent needs authentication, it will prompt the user to connect. To pre-authenticate users, see [Manual Authentication](/tool-router/manual-authentication).
```python
from agents import Agent, Runner

agent = Agent(
    name="Personal Assistant",
    instructions="You are a helpful personal assistant.",
    tools=session.tools()
)

result = Runner.run_sync(
    agent,
    "Summarize all the emails in my Gmail inbox today"
)
print(result.final_output)
```

To use OpenAI Agents SDK with Tool Router MCP instead, see [Using with MCP Clients](/tool-router/using-with-mcp-clients#openai).
</Step>
</Steps>

</Tab>

## What just happened?

When you ran this agent, Tool Router handled everything behind the scenes:

1. **[Search](/tool-router/tool-search)** - Found the right Gmail tools from 1000+ available tools
2. **[Authentication](/tool-router/authentication)** - Prompted you to connect your Gmail account
3. **[Execution](/tool-router/tool-execution)** - Fetched your emails and processed results efficiently

All you wrote was a prompt. Tool Router figured out the rest.
