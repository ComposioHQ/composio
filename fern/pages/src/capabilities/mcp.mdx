## Example

Add Composio to Claude Desktop's configuration:

```json claude_desktop_config.json
{
  "mcpServers": {
    "composio": {
      "command": "npx",
      "args": [
        "-y",
        "@composio/mcp-server",
        "your-composio-api-key"
      ]
    }
  }
}
```
