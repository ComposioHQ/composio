This guide walks you through **authenticated tool calling** â€” the foundation of
how Composio connects your AI agents to real-world actions. In this example, we
will use a simple [OpenAI](https://openai.com/) agent to perform tasks on behalf
of a user.

<Note>
    This guide assumes you have a basic understanding of AI agents and how they
    work. To use the SDK, you require an agentic provider. By default, SDK ships with
    [OpenAI](/docs/providers/openai) provider. You can learn more about different
    providers in our [Providers](/docs/providers) guide.
</Note>

<Steps>
    <Step title="Install the Composio SDK and OpenAI client">
        <CodeGroup>
        ```python Python
        pip install composio openai
        ```
        ```typescript TypeScript
        npm install @composio/core openai
        ```
        </CodeGroup>

        To initialize the Composio client, you will need API Key. You can find your
        Composio API key in the [Composio dashboard](https://platform.composio.dev/?next_page=/settings).
        
        <CodeGroup>
        ```python Python
        from composio import Composio
        from openai import OpenAI

        # Initialize the Composio client
        composio = Composio(api_key="<YOUR_COMPOSIO_API_KEY>")

        # Initialize the OpenAI client
        openai = OpenAI(api_key="<YOUR_OPENAI_API_KEY>")
        ```
        ```typescript TypeScript
        import { Composio } from '@composio/core';
        import { OpenAI } from 'openai';

        // Initialize the Composio client
        const composio = new Composio({ apiKey: "<YOUR_COMPOSIO_API_KEY>" });

        // Initialize the OpenAI client
        const openai = new OpenAI({ apiKey: "<YOUR_OPENAI_API_KEY>" });
        ```
        </CodeGroup>
    </Step>
    <Step title="Create a new connected account">
        <CodeGroup>
        ```python Python
        # Create a new connection request
        connection_request = composio.toolkits.authorize(toolkit="gmail")

        # Follow the redirect URL to authorize the connection
        print(connection_request.redirect_url)

        # Wait for the connection to be established
        connected_account = connection_request.wait_for_connection()
        print(connected_account)
        ```
        ```typescript TypeScript
        // Create a new connection request
        const connectionRequest = await composio.toolkits.authorize("gmail");

        // Follow the redirect URL to authorize the connection
        console.log(connectionRequest.redirectUrl);

        // Wait for the connection to be established
        const connection = await connectionRequest.waitForConnection();
        console.log(connection);
        ```
        </CodeGroup>
        <Warning>
            In this demo, we are using `composio.toolkits.authorize` to create a new
            connection request. This method is just for demo purposes. Refer to [Authentication](/docs/authentication)
            guide for more information on how to work with connected accounts.
        </Warning>
    </Step>
    <Step title="Run your agent">
        <CodeGroup>
        ```python Python
        # Fetch the tools for your agent
        tools = composio.tools.get("default",
            tools=["GMAIL_SEND_EMAIL"]
        )

        # Invoke agent
        completion = openai.chat.completions.create(
            model="gpt-4o",
            messages=[
                {
                    "role": "user",
                    "content": "say 'hi from the composio quickstart' to hey@composio.dev",
                },
            ],
            tools=tools,
        )
        # Handle Result from tool call
        result = composio.provider.handle_tool_calls(user_id=user_id, response=completion)
        print(result)
        ```
        ```typescript TypeScript
        // Fetch the tools for your agent
        const tools = await composio.tools.get("default", {
            tools: ["GMAIL_SEND_EMAIL"],
        });

        // Invoke agent
        const completion = await openai.chat.completions.create({
            model: "gpt-4o",
            messages: [
                {
                    "role": "user",
                    "content": "say 'hi from the composio quickstart' to hey@composio.dev",
                },
            ],
            tools: tools,
        });

        // Handle Result from tool call
        const result = await composio.provider.handleToolCalls("default", completion);
        console.log(result);
        ```
        </CodeGroup>
    </Step>
</Steps>

Congratulation! You have successfully integrated authenticated tool calling to your agent.
