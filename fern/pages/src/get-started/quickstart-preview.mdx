## Prerequisites

Before you begin, ensure you have:
- **Composio account**: [Sign up free](https://platform.composio.dev) (no credit card required)
- **API key**: Get it from [your dashboard](https://platform.composio.dev?next_page=/settings)
- **Development environment**: Python 3.10+ or Node.js 18+
- **AI provider API key**: OpenAI, Anthropic, or other supported provider


## Execute your first tool

Composio offers three ways to connect AI agents to tools:

<Tabs>
<Tab title="Tool Router">
Single MCP server and Tool that can search, authenticate, and execute across all Composio tools.

<Steps>
<Step title="Install SDK">
<CodeGroup>
```bash Python
pip install composio
```
```bash TypeScript
npm install @composio/core
```
</CodeGroup>
</Step>

<Step title="Create a session">
Initialize Tool Router with your toolkits:

<CodeGroup>
```python Python
from composio import Composio

composio = Composio()

session = composio.tool_router.create(
    user_id='user-123',
    toolkits=['gmail', 'calendar', 'slack', 'notion']
)

print(f'Session created: {session.id}')
```
```typescript TypeScript
import { Composio } from '@composio/core';

const composio = new Composio();

const session = await composio.toolRouter.create({
  userId: 'user-123',
  toolkits: ['gmail', 'calendar', 'slack', 'notion']
});

console.log('Session created:', session.id);
```
</CodeGroup>
</Step>

<Step title="Get tools and execute">
Use the session to get tools for any AI framework:

<CodeGroup>
```python Python
from openai import OpenAI

client = OpenAI()

tools = session.get_tools()

response = client.chat.completions.create(
    model="gpt-4",
    messages=[{
        "role": "user",
        "content": "Send an email to john@example.com about the meeting"
    }],
    tools=tools,
    tool_choice="auto"
)

result = session.handle_tool_call(response)
print(f"Result: {result}")
```
```typescript TypeScript
import OpenAI from 'openai';

const client = new OpenAI();

const tools = await session.getTools();

const response = await client.chat.completions.create({
  model: 'gpt-4',
  messages: [{

  tools: tools,

```

**Create an Auth Config for Gmail**

Auth Configs help you define how users should be authenticated.

<CodeGroup>
```python Python
from composio import Composio

composio = Composio()

# Create auth config for Gmail
auth_config = composio.auth_configs.create(
    toolkit="gmail",
    config={
        "client_id": "your_gmail_client_id",
        "client_secret": "your_gmail_client_secret",
        "scopes": ["https://www.googleapis.com/auth/gmail.send"]
    }
)

print(f"Auth Config ID: {auth_config.id}")
```
```typescript TypeScript
import { Composio } from '@composio/core';

const composio = new Composio();

// Create auth config for Gmail
const authConfig = await composio.authConfigs.create({
  toolkit: 'gmail',
  config: {
    clientId: 'your_gmail_client_id',
    clientSecret: 'your_gmail_client_secret',
    scopes: ['https://www.googleapis.com/auth/gmail.send']
  }
});

console.log(`Auth Config ID: ${authConfig.id}`);
```
</CodeGroup>

**Connect your users via the auth config**

Use composio auth links to let your users authenticate their accounts.

<CodeGroup>
```python Python
# Generate connection request for a user
connection_request = composio.connected_accounts.link(
    user_id="user-123",
    auth_config_id=auth_config.id
)

# Redirect user to this URL for authentication
print(f"Please authorize at: {connection_request.redirect_url}")

# Wait for connection to be established
connection = connection_request.wait_for_connection()
print(f"Connected: {connection.id}")
```
```typescript TypeScript
// Generate connection request for a user
const connectionRequest = await composio.connectedAccounts.link({
  userId: 'user-123',
  authConfigId: authConfig.id
});

// Redirect user to this URL for authentication
```

<Step title="Fetch tools and execute">
Get the tool your agent needs from composio and execute it.

<CodeGroup>
```python Python
from composio_openai import ComposioToolSet
import openai

# Initialize
openai_client = openai.Client()

# Fetch tools for the user
tools = toolset.get_tools(
    actions=["GMAIL_SEND_EMAIL"],
    entity_id="user-123"

# Use with OpenAI
response = openai_client.chat.completions.create(
    model="gpt-4",
    messages=[{
        "role": "user", 
        "content": "Send an email to john@example.com saying 'Meeting confirmed for 3pm tomorrow'"
    }],
    tools=tools,
    tool_choice="auto"
)

# Execute the tool call
result = toolset.handle_tool_call(response, entity_id="user-123")

```
```typescript TypeScript
import { OpenAIToolSet } from '@composio/openai';

// Initialize
```

<Step title="Create triggers and listen to events">
Setup triggers to run agents when specific events occur.

<CodeGroup>
```python Python
# Create a trigger for new Gmail messages
trigger = composio.triggers.create(
    user_id="user-123",
    toolkit="gmail",
    trigger_name="GMAIL_NEW_MESSAGE",
    config={
        "label_ids": ["INBOX"],
        "poll_interval": 60  # Check every 60 seconds
    }
)

print(f"Trigger created: {trigger.id}")

# Subscribe to trigger events
@composio.triggers.on("GMAIL_NEW_MESSAGE")
def handle_new_email(event):
    print(f"New email from: {event['from']}")
    print(f"Subject: {event['subject']}")
    
    # Process with AI
    response = openai_client.chat.completions.create(
        model="gpt-4",
        messages=[{
            "role": "user",
            "content": f"Summarize this email: {event['body']}"
        }]
    )
    
    print(f"Summary: {response.choices[0].message.content}")

# Start listening for events
composio.triggers.start_listening()
```
```typescript TypeScript
// Create a trigger for new Gmail messages
const trigger = await composio.triggers.create({
  userId: 'user-123',
  toolkit: 'gmail',
  triggerName: 'GMAIL_NEW_MESSAGE',
  config: {
    labelIds: ['INBOX'],
    pollInterval: 60  // Check every 60 seconds
  }
});

console.log(`Trigger created: ${trigger.id}`);

// Subscribe to trigger events
composio.triggers.subscribe('GMAIL_NEW_MESSAGE', async (event) => {
  console.log(`New email from: ${event.from}`);
  console.log(`Subject: ${event.subject}`);
  
  // Process with AI
  const response = await openai.chat.completions.create({
    model: 'gpt-4',
    messages: [{
      role: 'user',
      content: `Summarize this email: ${event.body}`
    }]
  });
  
  console.log(`Summary: ${response.choices[0].message.content}`);
});

// Start listening for events
await composio.triggers.startListening();
```
</CodeGroup>

<Note>
For production use, consider using webhooks instead of polling for better performance. See [Triggers documentation](/docs/using-triggers) for more details.
</Note>

```python
# Initialize
toolset = ComposioToolSet()
llm = ChatOpenAI(model="gpt-4")

# Create auth config for GitHub
auth_config = toolset.client.auth_configs.create(
    toolkit="github",
    config={"client_id": "your_id", "client_secret": "your_secret"}
)

# Connect user account
connection = toolset.client.connected_accounts.link(
    user_id="user-123",
    auth_config_id=auth_config.id
)
print(f"Authorize at: {connection.redirect_url}")
```
</Step>

<Step title="Execute tools">
```python
# Get tools for the user
tools = toolset.get_tools(actions=["GITHUB_CREATE_ISSUE"])

# Create agent
prompt = ChatPromptTemplate.from_messages([
    ("system", "You are a helpful assistant that can interact with GitHub."),
    ("human", "{input}"),
    ("placeholder", "{agent_scratchpad}")
])

agent = create_openai_functions_agent(llm, tools, prompt)
from langchain.agents import AgentExecutor
agent_executor = AgentExecutor(agent=agent, tools=tools, verbose=True)

# Execute
result = agent_executor.invoke({
    "input": "Create a new issue in the main repo about the bug we found"
}, config={"metadata": {"entity_id": "user-123"}})

print(f"Issue created: {result['output']}")
```
