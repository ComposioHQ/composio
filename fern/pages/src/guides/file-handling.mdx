If a tool requires a file to uploaded or downloaded, Composio SDK does this
automatically for you.

<CodeGroup>
    ```python Python
    from composio import Composio

    # Initialize the Composio client
    composio = Composio(api_key="<COMPOSIO_API_KEY>")

    # Execute the tool with a file
    response = composio.tools.execute(
        user_id="default",
        slug="GOOGLEDRIVE_UPLOAD_FILE",
        arguments={
            "file_to_upload": "path/to/file.txt",
        },
    )
    print(response)
    ```
    ```typescript TypeScript
    import { Composio } from "@composio/sdk";

    // Initialize the Composio client
    const composio = new Composio({apiKey: "<COMPOSIO_API_KEY>"});

    // Execute the tool with a file
    const response = await composio.tools.execute("GOOGLEDRIVE_UPLOAD_FILE", {
        userId: "default",
        arguments: {
            file_to_upload: "path/to/file.txt",
        },
    });
    console.log(response);
    ```
</CodeGroup>

When downloading files, the SDK writes the files to a temporary directory and returns
path to the file as part of the response.

<CodeGroup>
    ```python Python
    from composio import Composio

    # Initialize the Composio client
    composio = Composio(api_key="<COMPOSIO_API_KEY>")

    # Download the file
    response = composio.tools.execute(
        user_id="default",
        slug="GOOGLEDRIVE_DOWNLOAD_FILE",
        arguments={
            "file_id": "1234567890",
        },
    )
    print(response)
    ```
    ```typescript TypeScript
    import { Composio } from "@composio/sdk";

    // Initialize the Composio client
    const composio = new Composio({apiKey: "<COMPOSIO_API_KEY>"});

    // Download the file
    const response = await composio.tools.execute("GOOGLEDRIVE_DOWNLOAD_FILE", {
        userId: "default",
        arguments: {
            file_id: "1234567890",
        },
    });
    console.log(response);
    ```
</CodeGroup>

This will write the file `~/.composio/outputs`, You can control the output directory
by setting the `file_download_dir` parameter.

<CodeGroup>
    ```python Python
    from composio import Composio

    # Initialize the Composio client
    composio = Composio(
      api_key="<COMPOSIO_API_KEY>",
      file_download_dir="./outputs",
    )
    ```
    ```typescript TypeScript
    import { Composio } from "@composio/sdk";

    // Initialize the Composio client
    const composio = new Composio({
      apiKey: "<COMPOSIO_API_KEY>",
      fileDownloadDir: "./outputs",
    });
    ```
</CodeGroup>
