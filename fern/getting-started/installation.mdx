title: Installation
---

Composio provides Python and TypeScript SDKs as well as plugins with other LLM frameworks like CrewAI, LangChain, AutoGen.

## Install the SDK

<Tabs>
  <Tab title="Python">
Before installing the SDK, ensure you have Python 3.8+.
  {/* <Tabs> */}
  {/* <Tab title='uv'> */}
  <Steps>
  <Step title="Install uv">
    ```bash
curl -LsSf https://astral.sh/uv/install.sh | sh
    ```
    </Step>
    <Step title="Install Composio">
    ```bash
    uv add composio_core composio_openai
    ```
  {/* </Tab> */}
  {/* <Tab title='pip'>
    ```bash
    pip install composio_core
    ```
  </Tab> */}
  {/* </Tabs> */}
  </Step>
  <Step title="Install the relevant plugin">
    Depending on what LLM framework you're using, you'll need to install the relevant plugin.
    ```bash
    uv add composio_crewai      # For CrewAI
    uv add composio_langchain   # For LangChain
    ```
    </Step>
    </Steps>
  </Tab>
  <Tab title="TypeScript">
Before installing the SDK, ensure you have NodeJS 16+.
  <Steps>
  <Step title="Install Composio">
  <Tabs>
  <Tab title='pnpm'>
  ```bash
  pnpm add composio
  ```
  </Tab>
  <Tab title='npm'>
    ```bash
    npm install composio
    ```
  </Tab>
  <Tab title='yarn'>
    ```bash
    yarn add composio
    ```
  </Tab>
  </Tabs>
  </Step>
    > The package includes TypeScript type definitions out of the box.
  <Step title="Plugins">
  The TS package comes installed with support for:
  - [LangChain JS](/frameworks/javascript/langchain)
  - [Cloudflare Worker AI](/frameworks/javascript/cloudflare)
  - [Vercel AI SDK](/frameworks/javascript/vercel)
  </Step>
  </Steps>
  </Tab>
</Tabs>

## Get your API Key
Before you begin, you'll need a Composio account. Sign up [here](https://app.composio.dev) if you haven't yet! 
<Tabs>
<Tab title="CLI">
<Steps>
<Step title="Login">
<Tabs>
<Tab title="Python">

<Warning>Ensure you have `uv` installed!</Warning>

```bash
uvx --from composio-core composio login
```
</Tab>
<Tab title="Node">
```bash
npx composio-core login
```
</Tab>
</Tabs>
</Step>
<Step title="Verify">
To verify or check the authenticated account you can use the CLI
```bash
npx composio-core whoami
```
</Step>
<Step title="Store it securely">
When building, store the API key in an `.env` file.
```bash
echo "COMPOSIO_API_KEY=YOUR_API_KEY" >> .env
```

Or export it to your environment variables.

```bash
export COMPOSIO_API_KEY=YOUR_API_KEY
```

</Step>
</Steps>
</Tab>
<Tab title="Dashboard">
<Steps>
<Step title="Navigate to the Dasboard">
Navigate to [Settings > Project Settings](https://app.composio.dev/developers) and create an API key from the "Project API Keys" section.
</Step>
<Step title="Store it securely">
When building, store the API key in an `.env` file.
```bash
echo "COMPOSIO_API_KEY=YOUR_API_KEY" >> .env
```

Or export it to your environment variables.

```bash
export COMPOSIO_API_KEY=YOUR_API_KEY
```

</Step>
</Steps>
</Tab>
