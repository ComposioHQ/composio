<Tip>
 New App can be created on [Asana Developer Console](https://app.asana.com/0/my-apps).
</Tip>


Once you create the app, go to the "OAuth" tab and configure the below Redirect URL:

```
 https://backend.composio.dev/api/v1/auth-apps/asana/connection/oauth2/add
```

Once you added the Redirect URL, find your Client ID and Client Secret listed there and copy them. On Composio's [supported apps](https://app.composio.dev/apps) page, find Asana and click "Enable". Paste your Client ID and Secret, then Save to connect Asana.
