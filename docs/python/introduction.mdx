<Steps>
<Step title="Requirements">
  Ensure Python 3.8 or higher is installed on your system.
  You need pip available to install packages from PyPI.
</Step>
<Step title="Installation Steps">
  <CodeGroup>
  ```bash
  # Installing Core Package
  # Install the core Composio SDK
  pip install composio-core
  ```
  </CodeGroup>
  Install additional packages for specific frameworks as needed:
  <Tabs>
  <Tab title="Langchain">
    <CodeGroup>
      ```bash
      # Composio-Langchain installation
      # Installation for Langchain
      pip install composio-langchain
      ```
    </CodeGroup>
  </Tab>
  <Tab title="Autogen">
    <CodeGroup>
      ```bash
      # Composio-Autogen installation
      # Installation for Autogen
      pip install composio-autogen
      ```
    </CodeGroup>
  </Tab>
  <Tab title="CrewAI">
    <CodeGroup>
      ```bash
      # Composio-Crewai installation
      # Installation for CrewAI
      pip install composio-crewai
      ```
    </CodeGroup>
  </Tab>
  <Tab title="Lyzr">
    <CodeGroup>
      ```bash
      # Composio-lyzr installation
      # Installation for Lyzr
      pip install composio-lyzr
      ```
    </CodeGroup>
  </Tab>
  <Tab title="OpenAI">
    <CodeGroup>
      ```bash
      # Composio-Openai installation
      # Installation for OpenAI
      pip install composio-openai
      ```
    </CodeGroup>
  </Tab>
  <Tab title="Julep">
    <CodeGroup>
      ```bash
      # Composio-Julep installation
      # Installation for Julep
      pip install composio-julep
      ```
    </CodeGroup>
  </Tab>
  </Tabs>
</Step>

<Step title="Initializing the SDK">
<Warning>
    You should only initialize the SDK if you have not set up Composio via the CLI.
</Warning>
First, retrieve your API key from the [Composio dashboard](https://app.composio.dev/settings)
<CodeGroup>
     ```python
     # Initialise Composio directly with API Key
from composio import ComposioToolSet
composio_toolset = ComposioToolSet(api_key=*COMPOSIO_API_KEY*)
     ```
</CodeGroup>
</Step>

</Steps>
