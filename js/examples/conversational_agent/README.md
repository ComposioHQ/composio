# Conversational AI SDK Javascript example

## How to run the code

1. Clone this repo
2. `cd elevenlabs-examples/examples/conversational-ai`
3. Setup the environment variables
   - `cp .env.example .env`
   - Fill in the values
4. Install the dependencies

   ```bash
   npm install # install the dependencies
   ```

5. Run the script

   ```bash
   npm start
   ```

6. Visit localhost 
```html
http://localhost:3000/
```
