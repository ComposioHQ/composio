<Tip>OAuth app can be created after signing in your ClickUp account -> Integrations -> ClickUp API -> Create an App.</Tip>

Once you create the App, please configure the below Redirect URL:


```
https://backend.composio.dev/api/v1/auth-apps/add
```

After registering the app, copy the Client ID from the app page and generate/copy the Client Secret. On Composio's [Integrations](https://app.composio.dev/apps) page, find Clickup and add a new integration. Paste your Client ID and Secret, then Save to connect Clickup.
