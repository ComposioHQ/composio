---
title: "Create a new CLI session with auth code"
api: POST https://backend.composio.dev/api/v3/cli/create-session
description: "Generates a new CLI session with a random 6-character code. This endpoint is the first step in the CLI authentication flow, creating a session that can later be linked to a user account. The generated code is displayed to the user in the CLI and should be entered in the web interface to complete authentication."
contentType: "application/json"
---


### Headers

<ParamField header="x-api-key" type="string" required />

### Response

CLI session created successfully with a unique ID and temporary authentication code

<ResponseField name="id" type="string">
Unique identifier for the CLI session. UUID v4 format used for tracking and retrieval.
</ResponseField>

<ResponseField name="code" type="string">
The 6-character hexadecimal code used for CLI login

</ResponseField>

<ResponseField name="expiresAt" type="string">
The ISO timestamp when the session expires
</ResponseField>

<ResponseField name="status" type="enum">
The current status of the session

Allowed values: `pending` `linked`
</ResponseField>
