# CJS Example

This example demonstrates how to use Composio SDK with CommonJS.

## Setup

1. **Install dependencies:**
   ```bash
   pnpm install
   ```

2. **Configure environment:**
   ```bash
   cp .env.example .env
   ```
   
   Then edit `.env` and add your API keys:
   - `COMPOSIO_API_KEY`: Get it from [Composio Dashboard](https://app.composio.dev)

## Running the Example

```bash
# Run the example
pnpm start
```
